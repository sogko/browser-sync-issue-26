browser-sync-issue-26

## Install
```bash
git clone https://github.com/sogko/browser-sync-issue-26
cd browser-sync-issue-26
[sudo] npm install # some modules require sudo to install on some machines
```

## Run
1. Run expressjs server app
```bash
node app.js
```

2. Run gulp
```bash
gulp
```
